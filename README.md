### Algafood-API

### Empacotamento
```./mvnw package```

### Listar arquivos da pasta target
```ls target/```

### Executar aplicação
```java -jar target/algafood-api-0.0.1-SNAPSHOT.jar```

### Limpa os arquivos criados em builds anteriores
```./mvnw clean```

### Gerar arvore de dependências 
```./mvnw dependency:tree```

### Gerar arquivo pom
```./mvnw help:effective-pom```
